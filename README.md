videoplayer
===========


#一 简介#
------
此组件，主要是实现html5 video player 视频播放功能，主要实再了视频的通常的播放控制功能。
##1.1 主要功能##
- 播放暂停功能
- 进度条控制及拖放
- 音量控制
- 全屏控制，分为虚拟全屏和原生全屏
- ...
